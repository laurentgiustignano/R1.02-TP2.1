Introduction

Bienvenue dans cet exercice. Vous allez apprendre à séparer les préoccupations entre HTML et CSS.

Instructions

Le format de cette page n'est vraiment pas acceptable. On y retrouve du code CSS au milieu du texte HTML. C'est réellement compliqué à maintenir, et à faire évoluer. Pour corriger cela, il faut parcourir ce fichier, et en extraire les parties CSS pour les appliquer dans une feuille de style CSS.

Un exemple de code à ne pas faire :

<h2 style="text-decoration: underline">Instructions</h2>

Il faut absolument éviter ce genre de chose
